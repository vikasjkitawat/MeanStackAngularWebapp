## NodeJs

- Built on top of Chrome's V8 Engine
- JS anywhere outsode browsers
- Single Threaded
- Build JS Servers
- replaced all client side api with server side apis
- Server APIs
  - FS
  - HTTP
  - NET
  - Process

- Browser APIs (These are replaced by Server APIs)
  - XHR
  - DOM
  - Timer
